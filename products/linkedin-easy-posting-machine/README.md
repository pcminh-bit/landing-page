# LinkedIn Easy Posting Machine

**5 AI Skills** · Từ brainstorm nội dung đến post sẵn đăng trong ~27 phút  
**Dành cho:** Manager 30+, người dùng Claude / Cursor / bất kỳ AI agent đọc được `SKILL.md`

---

## Trong gói có gì

| Skill | Folder | Việc làm |
|-------|--------|----------|
| 1 · Ý tưởng | `linkedin-01-idea/` | 3 góc post theo avatar |
| 2 · Hook | `linkedin-02-hook/` | 3 hook (question / observation / contrarian) |
| 3 · Draft | `linkedin-03-draft/` | Bài nháp **300–400 từ** |
| 4 · Punch-line | `linkedin-04-punch-line/` | Polish punchy, **giữ 300–400 từ**, **không dùng `—`** |
| 5 · CTA | `linkedin-05-cta/` | CTA nhẹ + signature + p/s (tuỳ chọn) |

**Kèm theo:**

- `WORKSHEET-brand-voice.md` — điền trước bước 1  
- `WORKFLOW-27-phut.md` — checklist từng bước  
- `HUONG-DAN-SU-DUNG.md` — cài đặt chi tiết (in PDF nếu cần)  
- `assets/` — quy tắc chung (anti-AI-slop, hook, body, CTA)

## Bắt đầu nhanh

1. Điền `WORKSHEET-brand-voice.md`  
2. Mở agent → gọi skill 1 → copy block handoff sang skill tiếp theo  
3. Đọc `WORKFLOW-27-phut.md` lần đầu

**Trigger gợi ý:** `post lab bước 1` … `bước 5`, hoặc `@linkedin-01-idea/SKILL.md`

## Cài đặt (3 cách)

### Cách A — Cursor / Claude Code (khuyến nghị)

Copy cả thư mục `linkedin-easy-posting-machine` vào:

- **Cursor:** `~/.cursor/skills/` (mỗi skill là 1 folder con), **hoặc**
- Giữ nguyên cấu trúc bundle và `@` từng file `SKILL.md` trong project

### Cách B — Claude.ai (Projects)

Upload `SKILL.md` từng skill vào Project Knowledge, hoặc zip và đính kèm khi chat.

### Cách C — Agent khác

Đính kèm `SKILL.md` + `assets/` tương ứng vào context; paste block `=== SANG SKILL 0X ===` giữa các bước.

## FAQ

**Khác gì prompt pack?**  
Đây là **workflow 5 bước** — chỉnh được từng lớp (ý tưởng / hook / draft / punchy / CTA).

**Bài dài bao nhiêu từ?**  
Skill 3 và 4: **300–400 từ** (plain text, copy lên LinkedIn).

**Tại sao cấm `—`?**  
Em dash dễ lộ “vibe AI” trên LinkedIn VN. Dùng phẩy, chấm, hoặc xuống dòng.

**Bỏ bước 5 được không?**  
Được. Bài skill 4 đã đủ đăng nếu chưa cần CTA.

**Tiếng Anh?**  
Mặc định tiếng Việt; nói rõ trong prompt nếu muốn song ngữ.
