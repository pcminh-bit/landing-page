# Workflow ~27 phút

Điền **WORKSHEET-brand-voice.md** trước (5 phút). Phần còn lại ~22 phút nếu không chỉnh nhiều.

| Phút | Bước | Skill folder | Prompt mẫu |
|------|------|--------------|------------|
| 0–5 | Ý tưởng | `linkedin-01-idea` | `@linkedin-01-idea/SKILL.md` + chủ đề + worksheet |
| 5–8 | Hook | `linkedin-02-hook` | Paste `=== SANG SKILL 02 ===` |
| 8–18 | Draft | `linkedin-03-draft` | Paste `=== SANG SKILL 03 ===` |
| 18–24 | Punchy | `linkedin-04-punch-line` | Paste `=== SANG SKILL 04 ===` |
| 24–27 | CTA | `linkedin-05-cta` | Paste `=== SANG SKILL 05 ===` (hoặc bỏ qua) |

---

## Prompt mẫu — Bước 1

```text
@products/linkedin-easy-posting-machine/linkedin-01-idea/SKILL.md

Worksheet:
[paste 5 dòng đã điền]

Chủ đề: [1–3 câu tình huống / insight muốn đăng]
```

---

## Prompt mẫu — Bước 2–5

```text
@.../linkedin-02-hook/SKILL.md
[paste block === SANG SKILL 02 === từ output bước 1]
```

(Lặp lại với `03-draft`, `04-punch-line`, `05-cta` và block handoff tương ứng.)

---

## Checklist trước khi đăng

- [ ] 300–400 từ (sau bước 3 hoặc 4)  
- [ ] Không còn ký tự `—`  
- [ ] Plain text, không `##` / `---`  
- [ ] Đọc to 1 lần: có giống giọng bạn không?  
- [ ] CTA đúng mode (comment / link / không CTA)

---

## Lỗi thường gặp

| Lỗi | Cách xử lý |
|-----|------------|
| Agent viết luôn full post ở bước 1 | Nhắc: chỉ brainstorm, gọi skill 02 |
| Bài < 300 từ | Bảo skill 03 mở rộng 1 đoạn ví dụ |
| Còn `—` | Bảo skill 04 quét và thay hết |
| Generic / AI smell | Điền lại worksheet dòng 3–4 |
