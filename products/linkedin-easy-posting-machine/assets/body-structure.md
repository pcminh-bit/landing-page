# Cấu trúc body — Skill 03 (300–400 từ)

**Đếm từ:** tiếng Việt, gồm hook + body (chưa CTA/signature). Mục tiêu **300–400**; dưới 280 → thêm 1 đoạn insight; trên 420 → cắt filler.

## Khối (plain text, dòng trống giữa đoạn)

1. **Hook** — nguyên văn từ skill 02 (có thể sửa ≤10 từ cho khớp flow)
2. **Đoạn mở** — 2–4 câu: mở pain / situation (không lặp nguyên hook)
3. **Đoạn giữa** — insight hoặc story ngắn (1 ví dụ cụ thể — không tên khách; hoặc “một manager tôi từng…” nếu user cho phép generic)
4. **Đoạn lesson** — 1 takeaway actionable (manager 30+)
5. **Đoạn đóng** — 1–2 câu kết, **chưa** link; có thể kết bằng câu hỏi mở nhẹ (không trùng hook)

**Không** in nhãn `HOOK` / `BODY` ra output.

## Dạng post → nhịp body

| Dạng | Nhấn mạnh |
|------|-----------|
| story | 40% situation · 40% turning point · 20% lesson |
| insight | 30% observation · 50% why · 20% so-what |
| contrarian | 25% belief phổ biến · 50% reframe · 25% proof/logic |
| list | 3–4 ý, mỗi ý 1–2 câu (không đánh số 1.2.3. cứng nếu user thích flow prose) |
| case | before → shift → after (ẩn danh) |
| observation | 1 truth mở rộng 3 đoạn |
